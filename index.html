01 / MODEL DEFINITION

Hidden Markov Model

An HMM is defined by the parameter set \(\lambda = (A, B, \pi)\). At each time step, the system occupies a hidden state that generates an observable output.

π

Initial Probabilities

\[\pi_i = P(q_1 = i)\]

Belief that the system starts in state \(i\). All values sum to 1: \(\sum_i \pi_i = 1\).

A

Transition Probabilities

\[a_{ij} = P(q_{t+1} = j \mid q_t = i)\]

How the system evolves internally. Each row sums to 1: \(\sum_j a_{ij} = 1\).

B

Emission Probabilities

\[b_i(o) = P(O_t = o \mid q_t = i)\]

How hidden states generate visible observations. \(\sum_o b_i(o) = 1\).

💡
Analogy: Imagine listening to someone talk through a wall. You can hear the words (observations) but can't see their mood (hidden states). An HMM lets you guess their mood sequence from the words alone.

02 / E-STEP

Forward Probability α

The forward variable captures the joint probability of seeing the first \(t\) observations and being in state \(i\) at time \(t\).

Definition
\[\alpha_t(i) = P(O_1, O_2, \ldots, O_t,\; q_t = i \mid \lambda)\]
Initialization (t = 1)
\[\alpha_1(i) = \pi_i \cdot b_i(O_1)\]

Combines initial belief with compatibility of the first observation.

Recursion (t → t+1)
\[\alpha_{t+1}(j) = \left[\sum_{i=1}^{N} \alpha_t(i)\, a_{ij}\right] b_j(O_{t+1})\]

Depends on the entire past through αt and the current observation.

Worked Example — O = (W, H)

t=1

\(\alpha_1(R) = \pi_R \cdot b_R(W) = 0.6 \times 0.1 = 0.06\)

\(\alpha_1(S) = \pi_S \cdot b_S(W) = 0.4 \times 0.6 = 0.24\)

→
t=2

\(\alpha_2(R) = (0.06 \times 0.7 + 0.24 \times 0.4) \times 0.9 = (0.042 + 0.096) \times 0.9 = 0.1242\)

\(\alpha_2(S) = (0.06 \times 0.3 + 0.24 \times 0.6) \times 0.4 = (0.018 + 0.144) \times 0.4 = 0.0648\)

→
P(O|λ)

\(P(O \mid \lambda) = \sum_i \alpha_T(i) = 0.1242 + 0.0648 = 0.189\)

03 / E-STEP

Backward Probability β

The backward variable answers: if the system were in state \(i\) at time \(t\), how well could it explain all future observations?

Definition
\[\beta_t(i) = P(O_{t+1}, O_{t+2}, \ldots, O_T \mid q_t = i,\; \lambda)\]
Initialization (t = T)
\[\beta_T(i) = 1 \quad \forall i\]

No future observations: every state is equally compatible.

Recursion (t ← t+1)
\[\beta_t(i) = \sum_{j=1}^{N} a_{ij}\, b_j(O_{t+1})\, \beta_{t+1}(j)\]

Backward probability summarizes the entire future into a single value per state.

Worked Example — O = (W, H)

t=2

\(\beta_2(R) = 1,\quad \beta_2(S) = 1\)

←
t=1

\(\beta_1(R) = a_{RR}\,b_R(H)\,\beta_2(R) + a_{RS}\,b_S(H)\,\beta_2(S)\)

\(= 0.7 \times 0.9 \times 1 + 0.3 \times 0.4 \times 1 = 0.63 + 0.12 = 0.75\)

\(\beta_1(S) = 0.4 \times 0.9 + 0.6 \times 0.4 = 0.36 + 0.24 = 0.60\)

04 / SOFT CREDIT ASSIGNMENT

Auxiliary Variables γ and ξ

These variables perform soft credit assignment — distributing probability mass across all possible hidden state assignments.

State Responsibility γ
\[\gamma_t(i) = P(q_t = i \mid O, \lambda) = \frac{\alpha_t(i)\,\beta_t(i)}{P(O \mid \lambda)}\]

Fraction of belief that the system was in state \(i\) at time \(t\). Note: \(\sum_i \gamma_t(i) = 1\).

Transition Responsibility ξ
\[\xi_t(i,j) = \frac{\alpha_t(i)\,a_{ij}\,b_j(O_{t+1})\,\beta_{t+1}(j)}{P(O \mid \lambda)}\]

Fraction of probability flow through transition \(i \to j\) at time \(t\).

05 / M-STEP

Parameter Update Equations

Using the soft responsibilities computed in the E-step, we re-estimate model parameters as normalized expected counts.

π (Initial)
\[\hat{\pi}_i = \gamma_1(i)\]

Simply the responsibility at the very first time step.

A (Transition)
\[\hat{a}_{ij} = \frac{\displaystyle\sum_{t=1}^{T-1} \xi_t(i,j)}{\displaystyle\sum_{t=1}^{T-1} \gamma_t(i)}\]

Expected transitions \(i\!\to\!j\) divided by expected time in state \(i\).

B (Emission)
\[\hat{b}_i(o) = \frac{\displaystyle\sum_{t:\,O_t=o} \gamma_t(i)}{\displaystyle\sum_{t=1}^{T} \gamma_t(i)}\]

Expected times state \(i\) emitted symbol \(o\), normalized.

Emission Update — O = (W, H)

R totals

Total γ(R) = 0.238 + 0.657 = 0.895

γ(R) at W positions (t=1) = 0.238

\(\hat{b}_R(W) = 0.238/0.895 \approx 0.266\)

\(\hat{b}_R(H) = 0.657/0.895 \approx 0.734\)

+
S totals

Total γ(S) = 0.762 + 0.343 = 1.105

γ(S) at W positions (t=1) = 0.762

\(\hat{b}_S(W) = 0.762/1.105 \approx 0.689\)

\(\hat{b}_S(H) = 0.343/1.105 \approx 0.311\)

🔑 Key Insight

Observations do not update probabilities directly — they reshape belief flow, and belief flow updates the parameters. This is the core intuition of the Baum–Welch / EM approach.

08 / DECODING

Viterbi Algorithm

While Baum–Welch trains the model, Viterbi uses it — finding the single most probable hidden state sequence (the "Golden Path") for a given observation.

Viterbi Variable δ
\[\delta_t(j) = \max_{q_1,\ldots,q_{t-1}} P(q_1,\ldots,q_t=j,O_1,\ldots,O_t \mid \lambda)\]

Maximum probability of the most likely path ending in state \(j\) at time \(t\).

Recursion & Traceback
\[\delta_{t+1}(j) = \max_i[\delta_t(i)\,a_{ij}]\,b_j(O_{t+1})\]

Track argmax with \(\psi_{t+1}(j) = \arg\max_i[\delta_t(i)\,a_{ij}]\) for traceback.

09 / DEEP DIVE

Mathematical Deep Dive

①

Why EM Converges

Baum–Welch is guaranteed to never decrease \(\log P(O|\lambda)\). Each iteration finds parameters that satisfy:

\[P(O \mid \hat{\lambda}) \geq P(O \mid \lambda)\]

This monotonic improvement continues until convergence to a local maximum (EM does not guarantee global optima).

②

Scaling for Numerical Stability

Forward/backward probabilities decay exponentially with \(T\). To avoid underflow, use scaled variables:

\[\hat{\alpha}_t(i) = \frac{\alpha_t(i)}{c_t}, \quad c_t = \sum_i \alpha_t(i)\]

The log-likelihood is then \(\log P(O|\lambda) = -\sum_t \log c_t\).

③

Complexity

Per iteration of Baum–Welch:

\[\mathcal{O}(N^2 T)\text{ time},\quad \mathcal{O}(NT)\text{ space}\]

Where \(N\) = states, \(T\) = sequence length. The naive brute-force approach is \(\mathcal{O}(N^T)\) — exponential.

④

Multiple Observation Sequences

For \(K\) sequences \(\{O^{(k)}\}\), the update equations generalize by summing across sequences:

\[\hat{a}_{ij} = \frac{\sum_{k=1}^K \sum_{t=1}^{T_k-1} \xi_t^{(k)}(i,j)}{\sum_{k=1}^K \sum_{t=1}^{T_k-1} \gamma_t^{(k)}(i)}\]
